# Configuration du Style d'Écriture

## Instructions Générales

Ce fichier définit les paramètres stylistiques pour l'écriture du livre. Modifiez les valeurs selon vos préférences avant de commencer la rédaction. Les valeurs par défaut sont conçues pour un thriller dystopique adulte accessible.

---

## 1. Voix Narrative

### Point de Vue
- **Valeur** : `troisieme_personne_limitee`
- **Options** : `premiere_personne`, `troisieme_personne_limitee`, `troisieme_personne_omnisciente`
- **Description** : Perspective narrative du récit. Par défaut, focalisé sur Élias uniquement.

### Temps de Narration
- **Valeur** : `passe`
- **Options** : `present`, `passe`, `mixte`
- **Description** : Temps grammatical principal. Le passé crée une distance narrative classique.

### Voix du Narrateur
- **Ton** : `neutre_avec_touches_lyriques`
- **Options** : `neutre`, `neutre_avec_touches_lyriques`, `lyrique`, `sec_journalistique`, `poetique`
- **Description** : Caractère de la voix narrative. Des touches lyriques pour les moments de nature et d'émotion forte.

---

## 2. Niveau de Langage

### Registre Global
- **Valeur** : `standard_soutenu`
- **Options** : `familier`, `standard`, `standard_soutenu`, `litteraire`
- **Description** : Niveau de langue général du texte.

### Vocabulaire
- **Complexité** : `moyen`
- **Options** : `simple`, `moyen`, `riche`, `tres_riche`
- **Description** : Diversité et sophistication du vocabulaire.
- **Note** : Éviter le jargon excessif, privilégier la clarté pour l'immersion.

### Termes Techniques
- **Fréquence** : `modere`
- **Options** : `rare`, `modere`, `frequent`, `tres_frequent`
- **Description** : Utilisation de termes scientifiques/techniques liés à la dystopie.
- **Note** : Toujours contextualiser pour la compréhension.

---

## 3. Structure des Phrases

### Longueur Moyenne
- **Valeur** : `variee_courte_moyenne`
- **Options** : `courte`, `moyenne`, `longue`, `variee_courte_moyenne`, `variee_tous_types`
- **Description** : Phrases courtes pour l'action, moyennes pour la description, variation pour le rythme.

### Complexité Syntaxique
- **Valeur** : `moyen`
- **Options** : `simple`, `moyen`, `complexe`, `tres_complexe`
- **Description** : Utilisation de subordonnées, propositions enchâssées, etc.

### Rythme des Phrases
- **Valeur** : `variable_selon_scene`
- **Options** : `uniforme`, `variable_selon_scene`, `staccato`, `fluide`
- **Description** : Adapter le rythme à l'intensité de la scène.
  - Action/tension : phrases courtes, rythme rapide
  - Réflexion : phrases plus longues, rythme contemplatif
  - Description nature : phrases fluides, respirantes

---

## 4. Descriptions

### Densité Descriptive
- **Valeur** : `equilibree`
- **Options** : `minimaliste`, `legere`, `equilibree`, `riche`, `tres_detaillee`
- **Description** : Équilibre entre action et description.

### Focus Sensoriel
- **Vue** : `80%`
- **Ouïe** : `60%`
- **Odorat** : `40%`
- **Toucher** : `50%`
- **Goût** : `20%`
- **Description** : Importance relative de chaque sens (0-100%). Privilégier vue et ouïe, mais varier.

### Style de Description
- **Approche** : `impressionniste_emotionnelle`
- **Options** : `factuelle`, `impressionniste`, `impressionniste_emotionnelle`, `lyrique`, `symbolique`
- **Description** : Descriptions filtrées par l'état émotionnel d'Élias.

### Métaphores et Comparaisons
- **Fréquence** : `modere`
- **Options** : `rare`, `modere`, `frequent`, `tres_frequent`
- **Description** : Utilisation de figures de style.
- **Note** : Privilégier les métaphores liées à la nature, l'enfermement, la lumière/obscurité.

---

## 5. Dialogues

### Format
- **Valeur** : `guillemets_francais_tirets`
- **Options** : `guillemets_francais_tirets`, `guillemets_anglais`, `tirets_uniquement`
- **Description** : « Dialogue », —répondit-il.

### Naturalisme
- **Valeur** : `realiste_epure`
- **Options** : `tres_naturel_hesitations`, `realiste_epure`, `stylise`, `formel`
- **Description** : Dialogues réalistes mais sans tics verbaux excessifs.

### Incises Narratives
- **Fréquence** : `modere`
- **Options** : `rare`, `modere`, `frequent`
- **Description** : Utilisation de "dit-il", descriptions d'actions pendant le dialogue.

### Longueur des Répliques
- **Valeur** : `variee`
- **Options** : `courte`, `moyenne`, `longue`, `variee`
- **Description** : Varier selon le personnage et la situation.
  - Élias : répliques moyennes, réfléchies
  - Voss : plus longues, discours
  - Nora : plus courtes, directes

### Sous-texte
- **Importance** : `eleve`
- **Options** : `faible`, `moyen`, `eleve`, `tres_eleve`
- **Description** : Ce qui n'est pas dit est aussi important que ce qui est dit.

---

## 6. Rythme Narratif

### Vitesse Générale
- **Valeur** : `variable_par_acte`
- **Description** :
  - Acte I : Moyen, progression constante
  - Acte II : Rapide, tension croissante
  - Acte III : Lent, contemplatif, philosophique

### Équilibre Action/Réflexion
- **Acte I** : `40% action / 60% réflexion`
- **Acte II** : `70% action / 30% réflexion`
- **Acte III** : `30% action / 70% réflexion`

### Ellipses Temporelles
- **Fréquence** : `occasionnel`
- **Options** : `rare`, `occasionnel`, `frequent`
- **Description** : Utiliser avec parcimonie. Grande ellipse à la fin (6 mois).

### Scènes vs Sommaires
- **Ratio** : `70% scènes / 30% sommaires`
- **Description** : Privilégier les scènes dramatiques, résumer les transitions.

---

## 7. Atmosphère et Ton

### Ton Global
- **Valeur** : `sombre_avec_nuances`
- **Options** : `leger`, `neutre`, `sombre`, `sombre_avec_nuances`, `desespere`, `cynique`
- **Description** : Dystopique mais avec moments de beauté (nature) et d'espoir (puis désillusion).

### Émotions Dominantes
1. **Inquiétude/Malaise** : 70%
2. **Émerveillement** : 20% (nature, découvertes)
3. **Désespoir** : 40%
4. **Colère** : 30%
5. **Résignation** : 60% (crescendo vers la fin)

### Humour
- **Fréquence** : `rare_noir`
- **Options** : `absent`, `rare_noir`, `occasionnel`, `frequent`
- **Description** : Humour noir très occasionnel pour désamorcer la tension.

---

## 8. Éléments Stylistiques Spécifiques

### Flashbacks
- **Fréquence** : `3_a_5_dans_le_livre`
- **Intégration** : `italique_ou_separation_claire`
- **Focus** : Maya, moments clés du passé d'Élias
- **Longueur** : Courts (1-2 pages max), impactants

### Symbolisme
- **Utilisation** : `modere_subtil`
- **Symboles Clés** :
  - Lumière/Obscurité : Vérité vs mensonge
  - Haut/Bas : Hiérarchie sociale, vérité enfouie
  - Nature : Liberté perdue, beauté préservée
  - Enfermement : Physique et mental

### Répétitions Stylistiques
- **Approche** : `leitmotivs_subtils`
- **Exemples** :
  - "Le ciel qu'il n'avait jamais vu"
  - "Les chiffres ne mentent pas" (Élias au début) vs "Les chiffres ne disent pas tout" (fin)
  - Descriptions récurrentes des écrans de propagande

### Ponctuation Expressive
- **Utilisation** : `modere`
- **Description** :
  - Points de suspension : doutes, non-dits
  - Tirets : ruptures de pensée
  - Phrases courtes isolées : impact émotionnel
  - Pas d'excès : éviter les points d'exclamation multiples

---

## 9. Perspective et Distance Narrative

### Proximité Émotionnelle
- **Valeur** : `proche_filtree`
- **Options** : `tres_distante`, `distante`, `proche_filtree`, `tres_proche_intime`
- **Description** : Dans la tête d'Élias mais avec distance pour l'analyse.

### Discours Indirect Libre
- **Fréquence** : `frequent`
- **Options** : `rare`, `occasionnel`, `frequent`, `tres_frequent`
- **Description** : Mélange pensées d'Élias et narration pour fluidité.

### Introspection
- **Profondeur** : `elevee`
- **Options** : `minimale`, `moderee`, `elevee`, `tres_profonde`
- **Description** : Livre psychologique autant que thriller.

---

## 10. Contraintes et Choix Spécifiques

### Longueur des Chapitres
- **Moyenne** : `5 pages (4-6 pages)`
- **Variation** : Chapitres d'action peuvent être plus courts, chapitres contemplatifs plus longs

### Débuts de Chapitres
- **Style** : `in_medias_res_ou_accroche`
- **Description** : Commencer directement dans l'action ou avec une ligne percutante

### Fins de Chapitres
- **Style** : `cliffhanger_modere_ou_reflexion`
- **Description** : Alterner entre suspense et moment de réflexion selon le rythme

### Champs Lexicaux Récurrents
- Enfermement : murs, grilles, plafonds, limites
- Nature : ciel, vent, arbres, horizon
- Technologie : écrans, codes, données, surveillance
- Alimentation : rations, hydroponique, récoltes
- Lumière : néons, obscurité, aube, étoiles

---

## 11. Gestion des Révélations

### Rythme des Découvertes
- **Approche** : `progressif_avec_chocs`
- **Description** :
  - Indices semés dès le début
  - Révélations majeures : Chapitre 7 (complexe), Chapitre 14 (philosophie de Voss), Chapitre 16 (Maya)
  - Chaque révélation change la perspective

### Prévisibilité vs Surprise
- **Ratio** : `60% anticipable / 40% surprenant`
- **Description** : Le lecteur doit sentir venir certaines choses mais être surpris par d'autres (notamment l'ambiguïté morale de Voss)

---

## 12. Traitement des Thèmes

### Subtilité vs Explicite
- **Valeur** : `montre_plus_que_dit`
- **Options** : `tres_explicite`, `equilibre`, `montre_plus_que_dit`, `entierement_subtil`
- **Description** : Thèmes explorés à travers l'action et les dilemmes, pas de moralisation directe

### Dialogues Philosophiques
- **Fréquence** : `concentre_acte_III`
- **Style** : `naturel_dans_conflit`
- **Description** : Débat Élias/Voss doit sembler organique, pas comme un essai philosophique

---

## 13. Sensibilité et Contenu

### Violence
- **Niveau** : `suggere_impact_psychologique`
- **Options** : `hors_scene`, `suggere_impact_psychologique`, `decrit_moderement`, `graphique`
- **Description** : Conditions inhumaines montrées mais sans complaisance gore

### Contenu Mature
- **Langage** : `quelques_jurons_contextualises`
- **Sexualité** : `absent_ou_tres_suggere`
- **Description** : Roman adulte mais focus sur le thriller psychologique

### Désespoir
- **Traitement** : `realiste_pas_gratuit`
- **Description** : Montrer la dureté sans sombrer dans le nihilisme gratuit

---

## 14. Cohérence et Continuité

### Suivi des Détails
- **Rigueur** : `elevee`
- **Description** : Tracker les détails (apparence personnages, géographie, dates) pour cohérence

### Voix des Personnages
- **Distinction** : `claire_et_consistante`
- **Élias** : Analytique, phrases structurées, questions internes
- **Voss** : Rhétorique, persuasif, phrases amples
- **Nora** : Directe, technique, concise
- **Morgenstern** : Clinique, détaché, précis

---

## 15. Objectifs Émotionnels par Acte

### Acte I
- **Cible** : Curiosité, inquiétude croissante, connexion avec Élias

### Acte II
- **Cible** : Choc, horreur, tension, désespoir, puis détermination

### Acte III
- **Cible** : Confusion morale, questionnement, malaise, résignation troublante

### Objectif Final
- **Cible** : Laisser le lecteur troublé, questionnant ses propres valeurs, sans réponse facile

---

## Notes pour l'Adaptation

Ces paramètres peuvent être ajustés à tout moment. Après avoir lu les premiers chapitres, n'hésitez pas à :
- Modifier les valeurs
- Ajouter des instructions spécifiques
- Préciser des préférences stylistiques
- Demander des ajustements tonals

**L'écriture commencera une fois ce fichier validé et ajusté selon vos préférences.**

---

## Checklist de Validation

Avant de commencer l'écriture, vérifier :
- [ ] Le point de vue narratif convient
- [ ] Le niveau de langage correspond aux attentes
- [ ] L'équilibre action/réflexion par acte est approprié
- [ ] Les choix stylistiques (métaphores, descriptions) sont clairs
- [ ] Les objectifs émotionnels sont compris
- [ ] Le ton global correspond à la vision du livre
