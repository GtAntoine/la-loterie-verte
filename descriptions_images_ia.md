# Descriptions pour Génération d'Images IA - La Loterie Verte

## Style Général et Paramètres

### Style Artistique
- **Type** : Illustration réaliste avec touche dystopique
- **Ambiance** : Sombre, cinématographique, oppressante mais avec moments d'espoir
- **Palette** : Dominante gris-bleu-vert métallique pour les scènes dans les Murs ; verts naturels et ciel bleu pour le Dehors
- **Éclairage** : Néons froids pour l'intérieur, lumière naturelle chaude pour l'extérieur
- **Format** : Portrait vertical (format page de livre), ratio 2:3

### Paramètres IA Recommandés
```
Style: Digital illustration, cinematic, dystopian sci-fi, high detail
Quality: High resolution, professional book illustration
Mood: Dark, atmospheric, realistic with subtle stylization
Avoid: Cartoonish, overly colorful, fantasy elements
```

## NOTES D'UTILISATION

### Pour générer les images de personnages :
1. Utiliser d'abord les prompts des personnages principaux (PARTIE 1)
2. Sauvegarder ces images comme références
3. Utiliser ces références lors de la génération des images de chapitres

### Pour générer les images de chapitres :
1. Utiliser les prompts de la PARTIE 3
2. Référencer les ID de personnages appropriés (ex: [ELIAS-01])
3. Référencer les ID d'environnements si nécessaire (ex: [ENV-03])
4. Maintenir le format portrait vertical pour cohérence de livre

### Cohérence stylistique :
- Tous les prompts incluent "cinematic dystopian illustration" ou variante
- Palette cohérente : gris-bleu froid pour les Murs, vert-bleu naturel pour le Dehors
- Éclairage cohérent : néons froids vs lumière naturelle chaude
- Niveau de détail réaliste mais légèrement stylisé

### Ordre de génération recommandé :
1. Personnages principaux (Élias, Nora, Voss, Maya, Kaelin)
2. Environnements clés (Murs, Complexe, Villa, Dehors)
3. Images de chapitres dans l'ordre (1-16)
